# Telegram Bot Project

This project is a simple Telegram bot built using Python. It serves as an example of how to create and manage a Telegram bot, including command handling and user interaction.

## Project Structure

```
telegram-bot-project
├── src
│   ├── bot.py          # Main entry point for the Telegram bot
│   └── utils.py        # Utility functions for the bot
├── requirements.txt     # List of dependencies
└── README.md            # Project documentation
```

## Setup Instructions

1. Clone the repository:
   ```
   git clone <repository-url>
   cd telegram-bot-project
   ```

2. Install the required dependencies:
   ```
   pip install -r requirements.txt
   ```

3. Set up your Telegram bot by talking to the [BotFather](https://t.me/botfather) on Telegram and obtain your bot token.

4. Update the bot token in `src/bot.py`.

## Usage

To run the bot, execute the following command:
```
python src/bot.py
```
